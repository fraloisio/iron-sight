Iron Sight — Connection Guide

Pi hostname: fraspberry2  ·  Three ways to connect

Option A

Normal WiFi

Use this at the museum or anywhere with the ICO-860CAD network.

  1. Connect your Mac to ICO-860CAD WiFi.
  2. Pi connects automatically on boot.
  3. SSH in: ssh admin@fraspberry2.local

Option B

Pi Hotspot Fallback

WiFi not available. Pi creates its own network after 30s on boot, or manually.

  1. Wait ~30s after Pi boots with no WiFi — hotspot starts automatically.
  2. Or SSH in via USB and run: sudo ironside-hotspot.sh
  3. On your Mac, connect to WiFi: ironside / password: ironsight
  4. SSH in: ssh admin@10.42.0.1

Option C

USB Cable (no network needed)

Last resort. Direct Mac↔Pi connection via USB data cable. No WiFi required.

  1. Use a data USB cable (not charge-only) — micro USB into the Pi's USB port (not PWR).
  2. Make sure Internet Sharing is OFF on your Mac (System Settings → General → Sharing).
  3. Mac USB Gadget interface should be set to Using DHCP.
  4. Wait ~30s for Pi to boot and assign your Mac an IP.
  5. SSH in: ssh admin@10.12.194.1

cd ~/pi-scripts && python3 ir_detect.py
